Localized chatting app

## Installation and Building

```bash
# clone repository
git clone https://github.com/Hello-Neighbor/Hello-Neighbor-Chat.git
cd Hello-Neighbor-Chat

# install required packages and dependencies
npm install

# build and run locally
npm run dev

```
